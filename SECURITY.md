# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.3.x   | :white_check_mark: |
| < 1.3   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in SimOffice, please report it responsibly.

**Email:** public@valuecell.ai

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant logs or screenshots

We will acknowledge your report within 48 hours and provide an update within 7 days. If the vulnerability is accepted, we will work on a fix and credit you in the release notes (unless you prefer to remain anonymous).

Please do not open a public GitHub issue for security vulnerabilities.
