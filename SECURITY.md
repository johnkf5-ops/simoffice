# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in SimOffice, please report it responsibly via [GitHub Security Advisories](https://github.com/johnkf5-ops/simoffice/security/advisories/new).

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant logs or screenshots

We will acknowledge your report within 48 hours and provide an update within 7 days. If the vulnerability is accepted, we will work on a fix and credit you in the release notes (unless you prefer to remain anonymous).

Please do not open a public GitHub issue for security vulnerabilities.
